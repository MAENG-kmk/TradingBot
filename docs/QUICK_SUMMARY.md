# TradingBot 프로젝트 빠른 요약

## 🎯 한 문장 요약
**Binance 선물에서 볼린저 밴드 + MACD로 4시간 봉 기반 자동 매매하는 봇**

---

## 📊 핵심 정보

| 항목 | 내용 |
|------|------|
| **거래소** | Binance Futures |
| **전략** | 볼린저 밴드 + MACD |
| **타임프레임** | 4시간 봉 |
| **포지션 크기** | 계정의 10% × 최대 10개 |
| **목표 수익** | +5% |
| **손절** | -2% |
| **체크 주기** | 15초 |
| **방향** | 롱/숏 양방향 |

---

## 🔄 실행 흐름 (15초마다)

```
1. 포지션 조회
2. 청산 체크 (목표 +5% or 손절 -2%)
3. 잔고 확인
4. 진입 체크
   ├─ 모든 코인 스캔
   ├─ 볼린저 + MACD 신호 확인
   └─ 조건 만족 시 진입
5. 15초 대기
```

---

## 📂 핵심 파일

```
main.py                  # 메인 루프 ⭐
tools/BetController.py   # 포지션 관리 ⭐⭐⭐
logics/enterPosition.py  # 진입 로직 ⭐⭐⭐
logics/closePosition.py  # 청산 로직 ⭐⭐⭐
```

---

## 🎯 전략

**진입**: 볼린저 하단 돌파 + MACD 골든크로스 → 롱  
**청산**: +5% 목표 또는 -2% 손절
